About Us

Welcome to BootStrap!

  • UBootstrap is a popular front-end development framework that provides a set of tools and components for building responsive websites and web applications. Developed by Twitter, Bootstrap has become a go-to framework for web developers because of its ease of use, flexibility, and robust features.
  • One of the key advantages of Bootstrap is its grid system. The grid system allows developers to create responsive layouts that adapt to different screen sizes and devices. The grid system is based on a 12-column layout, which can be easily customized and configured. This makes it easy for developers to create responsive designs that look great on desktops, tablets, and smartphones.
  • Another advantage of Bootstrap is its wide range of pre-built components. Bootstrap comes with a set of UI components that can be easily customized to fit the needs of a specific project. These components include navigation menus, forms, buttons, alerts, modals, and much more. Using these pre-built components can save a lot of time and effort for developers, allowing them to focus on the core functionality of their project.
  • Bootstrap also provides a rich set of JavaScript plugins that add functionality and interactivity to websites and web applications. These plugins include carousels, tooltips, popovers, modals, and much more. These plugins are easy to use and can be customized to fit the needs of a specific project.

Bootstrap is also highly customizable. Developers can easily customize the color scheme, typography, and other design elements of Bootstrap to fit their branding and design requirements. Additionally, developers can use the Sass preprocessor to create custom stylesheets that extend Bootstrap's core styles.

  • Overall, Bootstrap is a powerful and flexible framework that makes it easy for developers to create responsive websites and web applications. Its wide range of pre-built components, grid system, and JavaScript plugins make it a go-to choice for developers who want to save time and effort while creating great-looking websites and web applications.

Build and extend in real-time with CSS variables

Bootstrap 5 is evolving with each release to better utilize CSS variables for global theme styles, individual components, and even utilities. We provide dozens of variables for colors, font styles, and more at a :root level for use anywhere. On components and utilities, CSS variables are scoped to the relevant class and can easily be modified.

  • 01 Using CSS variables

    Use any of our global :root variables to write new styles. CSS variables use the Var(--bs-variableName) syntax and can be inherited by children elements.

    .component {

    color: var(--bs-gray-800);

    background-color: var(--bs-gray-100);

    border: 1px solid var(--bs-gray-200);

    border-radius: .25rem;

    }

    .component-header {

    color: var(--bs-purple);

    }

  • 02Customizing via CSS variables

    Override global, component, or utility class variables to customize Bootstrap just how you like. No need to redeclare each rule, just a new variable value.

    body {

    --bs-body-font-family: var(--bs-font-monospace);

    --bs-body-line-height: 1.4;

    --bs-body-bg: var(--bs-gray-100);

    }

    .table {

    --bs-table-color: var( --bs-gray-600);

    --bs-table-bg: var(--bs-gray-100);

    --bs-table-border-color: transparent;

    }

  • 03 Create and extend utilities

    Use Bootstrap's utility API to modify any of our included utilities or create your own custom utilities for any project. Import Bootstrap first, then use Sass map functions to modify, add, or remove utilities.

    @import "bootstrap/scss/bootstrap";

    $utilities: map-merge(

    $utilities,

    (

    "cursor": (

    property:cursor,

    class:cursor,

    responsive: true,

    values: auto pointer grab,

    )

    )

    );
